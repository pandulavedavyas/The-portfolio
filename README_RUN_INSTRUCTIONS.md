## 🔧 Customization

### To Change Your Information:

1. Open `portfolio.html` in any text editor
2. Search for the text you want to change
3. Replace it with your information
4. Save the file
5. Refresh your browser (F5)

**Example:**
- Find: `pandulavedavyas@gmail.com`
- Replace with: `your-email@example.com`

### To Change Colors:

In `portfolio.css`, find the `:root` section and modify:
```css
:root {
    --primary-blue: #00d4ff;    /* Blue glow */
    --primary-red: #ff006e;     /* Red accent */
    --primary-gold: #ffd60a;    /* Gold accent */
}
```

Find the "PROJECT ARCHIVE" section in `portfolio.html` and add/edit project cards with your actual projects.
